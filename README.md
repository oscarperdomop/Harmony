## ğŸš€ Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [Git](https://git-scm.com/)


### Installation

1. Clone the repo

```bash

git clone https://github.com/JsebasC/Harmony.git

```

2. Go to project folder

```bash

cd harmony

```

3. Install packages

```bash

npm install

```
